Run Syntax
----------
npm run baseline -- --flow=<FLOW> --bu=<BU> --url=<URL> --tc=<TC> --project=<BROWSER>

Flow:
  --flow=1      New Entity - Modify
  --flow=2      Existing Entity - Modify
  --flow=3      New Entity - View
  --flow=4      Existing Entity - View
  --flow=1,2    Modify flows only
  --flow=3,4    View flows only
  --flow=all    All four flows

KPS Commands
------------
KPS, Modify flows, all test cases TC01 to TC05:
  npm run baseline -- --flow=1,2 --bu=KPS --url=DPBH --tc=all --project=chromium

KPS, New Entity - Modify, all test cases TC01 to TC05:
  npm run baseline -- --flow=1 --bu=KPS --url=DPBH --tc=all --project=chromium

KPS, Existing Entity - Modify, all test cases TC01 to TC05:
  npm run baseline -- --flow=2 --bu=KPS --url=DPBH --tc=all --project=chromium

KPS, View flows, all test cases TC01 to TC05:
  npm run baseline -- --flow=3,4 --bu=KPS --url=DPBH --tc=all --project=chromium

KPS, TC05 only:
  npm run baseline -- --flow=all --bu=KPS --url=DPBH --tc=TC05 --project=chromium

KPS, headed mode:
  npm run baseline -- --flow=all --bu=KPS --url=DPBH --tc=all --project=chromium --headed

KPS, Existing Entity - Modify with temporary entity override:
  npm run baseline -- --flow=2 --bu=KPS --url=DPBH --entity="YOUR_FACILITY_NAME" --tc=all --project=chromium

KPS, Existing Entity - View with temporary entity override:
  npm run baseline -- --flow=4 --bu=KPS --url=DPBH --entity="YOUR_FACILITY_NAME" --tc=all --project=chromium


DPBH Commands
-------------
All DPBH BUs, all four flows, all test cases:
  npm run baseline -- --flow=all --bu=HF,ML,EHS,CCP,KPS --url=DPBH --tc=all --project=chromium

All DPBH BUs, TC05 only:
  npm run baseline -- --flow=all --bu=HF,ML,EHS,CCP,KPS --url=DPBH --tc=TC05 --project=chromium

  npm run baseline -- --flow=all --bu=all --url=DPBH --tc=all --project=chromium
